# Free Roam Unity Project 🎮

## Project Description

Free Roam, collect coins and jump.

## Controls

- **Horizontal Move:** W,A,S,D
- **Direction:** Use Mouse
- **Jumo:** Spacebar (can double jump)

✔️ Double Jump
✔️ Direction
✔️ Collect Coins
✔️ Increase Score
